# DataPulse LeadGen 📈

> **A modern fintech‑style lead generation dashboard** built with Python, Flask, and a sleek CSS UI.  
> Perfect for showcasing data‑driven insights in a portfolio‑ready, production‑grade web app.

---

## ✨ Overview

DataPulse LeadGen provides an interactive dashboard that visualizes lead‑generation metrics in real‑time.  
It combines a lightweight Flask backend, a responsive CSS design, and dynamic HTML templates to deliver a polished user experience.

## ✨ Features

- **Real‑time charts** powered by Chart.js (or your preferred library).
- **Fintech‑inspired dark/glassmorphism UI** with smooth micro‑animations.
- **Responsive layout** – looks great on desktop and mobile.
- **Easy data ingestion** – connect your CSV/JSON sources with a simple Python service.
- **Extensible architecture** – add new widgets or data sources without touching core code.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Python 3.11, Flask |
| Frontend | HTML5, CSS3 (custom design), optional JS libraries |
| Styling | Modern dark mode, glass‑morphism, gradients |
| Version Control | Git |

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/yourusername/DataPulse-LeadGen.git
   cd DataPulse-LeadGen
   ```
2. **Create a virtual environment**
   ```bash
   python -m venv venv
   .\\venv\\Scripts\\activate   # Windows
   ```
3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the app**
   ```bash
   python app.py
   ```
   Open `http://127.0.0.1:5000` in your browser.

## 📊 Usage

- Navigate to the dashboard to explore lead‑generation metrics.
- Edit `static/style.css` to tweak colors or animations.
- Extend the Flask routes in `app.py` for new data sources.
